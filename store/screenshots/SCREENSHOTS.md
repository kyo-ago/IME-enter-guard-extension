## 撮影方法

### 1. HTMLファイルを開く
```bash
open store/screenshots/screenshot1_ja.html
```

### 2. DevToolsでスクリーンショット撮影

1. **F12** でDevToolsを開く
2. **Cmd + Shift + P** (Mac) / **Ctrl + Shift + P** (Windows)
3. `screenshot` と入力
4. **「Capture node screenshot」** を選択
5. Elementsタブで `.screenshot` 要素を選択

### 3. 出力サイズ
- 1280 x 800 px（Chrome Web Store推奨サイズ）

---

## Chrome Web Storeへのアップロード

1. [Developer Dashboard](https://chrome.google.com/webstore/devconsole) にアクセス
2. 拡張機能を選択 → 「ストアの掲載情報」
3. 「スクリーンショット」セクションで画像をアップロード
4. 最大5枚までアップロード可能

### 推奨構成
- **日本語ストア**: screenshot1_ja + screenshot2_ja
- **英語ストア**: screenshot1_en + screenshot2_en
- **中国語ストア**: screenshot1_zh_CN + screenshot2_zh_CN
