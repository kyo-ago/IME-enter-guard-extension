# IME Enter Guard

## 개요
IME(입력기)로 한글 변환 확정 시 Enter 키를 눌렀을 때 의도치 않은 폼 전송을 방지하는 확장 프로그램입니다.

## 문제점
한글 입력 시 글자를 확정하기 위해 Enter 키를 누르면, 많은 웹 애플리케이션(채팅 앱, 검색창, 폼 등)이 이를 "전송"으로 해석합니다. 이로 인해 입력 도중에 메시지가 전송되는 문제가 발생합니다.

## 해결책
IME Enter Guard는 IME 변환 중임을 감지하고 Enter 키 이벤트가 웹 애플리케이션에 전달되는 것을 차단합니다. 이를 통해:
- 자연스럽게 변환을 확정할 수 있습니다
- 의도치 않은 전송 없이 입력을 계속할 수 있습니다
- 전송 타이밍을 완전히 제어할 수 있습니다

## 특징
- 모든 웹사이트에서 자동으로 작동
- 설정 불필요
- 가볍고 빠름
- 데이터 수집 없음, 외부 통신 없음
- 오픈 소스

## 지원 입력 필드
- 텍스트 영역
- 텍스트 입력 필드

## 작동 원리
키보드 이벤트를 모니터링하고 IME 조합 상태를 감지합니다. 변환 중에 Enter가 눌리면 웹사이트로의 이벤트 전파를 중지하면서도 IME의 변환 확정은 정상적으로 이루어집니다.

## 개인정보
이 확장 프로그램은:
- 어떠한 데이터도 수집하지 않습니다
- 네트워크 통신을 하지 않습니다
- 특별한 권한을 필요로 하지 않습니다
- 브라우저 내에서 로컬로 작동합니다
