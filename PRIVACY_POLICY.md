# Privacy Policy / プライバシーポリシー

**Last Updated / 最終更新: 2025-01-02**

---

## English

### Overview
IME Enter Guard is a browser extension that prevents accidental form submissions when using IME (Input Method Editor). This privacy policy explains how we handle your data.

### Data Collection
**We do not collect any data.**

This extension:
- Does NOT collect personal information
- Does NOT collect browsing history
- Does NOT collect form input data
- Does NOT collect usage statistics
- Does NOT use analytics or tracking

### Data Storage
This extension does not store any data locally or remotely. It has no settings, no user accounts, and no data persistence of any kind.

### Network Requests
This extension makes NO network requests. It operates entirely offline and does not communicate with any external servers.

### Permissions
This extension requires minimal permissions:
- **Active Tab / All URLs**: Required to inject the content script that monitors keyboard events on web pages. This is necessary for the core functionality of preventing accidental submissions.

### Third-Party Services
This extension does not use any third-party services, libraries that collect data, or external APIs.

### Changes to This Policy
If we make changes to this privacy policy, we will update the "Last Updated" date above.

### Contact
For questions about this privacy policy, please open an issue on our GitHub repository.
